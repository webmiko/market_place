97 def add_product(self, product: "Product") -> None: 

98 """Добавляет продукт в категорию. 

99 

100 Метод добавляет объект класса Product или его наследников в приватный список 

101 товаров категории и увеличивает счетчик общего количества продуктов. 

102 

103 Метод защищен от добавления объектов, не являющихся продуктами. При попытке 

104 добавить не-продукт выбрасывается TypeError. 

105 

106 Метод также предотвращает добавление дубликатов - продуктов с такими же 

107 атрибутами. При попытке добавить дубликат выбрасывается ValueError. 

108 

109 Args: 

110 product: Объект класса Product или его наследников (Smartphone, LawnGrass и т.д.) 

111 для добавления в категорию 

112 

113 Raises: 

114 TypeError: Если product не является экземпляром класса Product или его наследников 

115 ValueError: Если продукт с такими же атрибутами уже существует в категории 

116 

117 Example: 

118 >>> product = Product("Test", "Description", 100.0, 5) 

119 >>> category = Category("Test Category", "Description", []) 

120 >>> initial_count = Category.product_count 

121 >>> category.add_product(product) 

122 >>> assert len(category._Category__products) == 1 

123 >>> assert Category.product_count == initial_count + 1 

124 

125 >>> from src.product import Smartphone 

126 >>> smartphone = Smartphone("Phone", "Desc", 100.0, 5, 95.5, "Model", 256, "Black") 

127 >>> category.add_product(smartphone) 

128 >>> assert len(category._Category__products) == 2 

129 

130 >>> category.add_product("Not a product") # doctest: +SKIP 

131 Traceback (most recent call last): 

132 ... 

133 TypeError: Можно добавлять только объекты класса Product и его наследников 

134 

135 >>> category.add_product(product) # doctest: +SKIP 

136 Traceback (most recent call last): 

137 ... 

138 ValueError: Продукт с такими же атрибутами уже существует в категории

140 if not isinstance(product, Product):
