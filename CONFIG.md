# Конфигурация проекта

## Длина строки кода: 119 символов

Проект использует единую длину строки кода **119 символов** для всех инструментов форматирования и линтинга.

### Настройки в конфигурационных файлах:

#### 1. `.flake8`
```ini
[flake8]
max-line-length = 119
```

#### 2. `pyproject.toml` - Black
```toml
[tool.black]
line-length = 119
```

#### 3. `pyproject.toml` - isort
```toml
[tool.isort]
line_length = 119
```

#### 4. `.editorconfig`
```ini
[*.py]
max_line_length = 119
```

### Проверка настроек

Убедитесь, что все инструменты используют правильную длину строки:

```bash
# Проверка форматирования
poetry run black --check .

# Проверка линтинга
poetry run flake8 .

# Форматирование кода
poetry run black .
poetry run isort .
```

### Использование через Makefile

```bash
# Форматирование (использует настройки из pyproject.toml)
make format

# Линтинг (использует настройки из .flake8)
make lint

# Все проверки
make check
```

### Примечания

- **mypy** не проверяет длину строки кода (это делают black и flake8)
- Все инструменты автоматически читают настройки из соответствующих конфигурационных файлов
- При использовании `poetry run` настройки берутся из `pyproject.toml`
- `.editorconfig` помогает IDE автоматически применять настройки длины строки
